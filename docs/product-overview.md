# Simulador de Juros Compostos

Aplicação web para simulação de investimentos com juros compostos, aportes mensais e análise de crescimento patrimonial.

## Objetivos

- Simular crescimento de investimentos mês a mês
- Visualizar evolução patrimonial
- Comparar cenários de investimento
- Considerar inflação e imposto
- Fornecer análise visual através de gráficos

## Público alvo

- Investidores iniciantes
- Pessoas planejando aposentadoria
- Usuários interessados em educação financeira

## Proposta de valor

- Interface simples
- Simulação rápida
- Visualização clara da evolução do patrimônio
- Comparação de cenários financeiros
